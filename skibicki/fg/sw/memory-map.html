Building memory map…
Stack: set SP at $E000 so pushes grow down into high work RAM Protected RAM: only 64 bytes, reserve for credits, latches, and tiny bookkeeping. Magic: reads from $0000-$3FFF are ROM reads; writes are graphics operations.

Robby Roto
Building memory map…
Stack: set SP at $0000 so pushes grow down into high work RAM Protected RAM: protected window moved up to $E000-$E1FF NVRAM: Robby Roto keeps NVRAM/protected storage at $E000-$E7FF while $8000-$DFFF is used as 24 KB of upper ROM Magic: reads from $0000-$3FFF are ROM reads; writes are graphics operations.
