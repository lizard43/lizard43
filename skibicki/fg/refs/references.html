This page is a catalog for the local reference folder. The file list is stored as JSON inside this HTML page, then rendered by JavaScript so the list can stay tidy without hand-writing a long stack of links.

Keep the files themselves in refs/. Each JSON entry only needs a title and a link. The page automatically detects the file extension and assigns a small file-type icon.
