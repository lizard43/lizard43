Color map

Color byte decode: upper 5 bits are NTSC phase/hue; lower 3 bits are luminance/color-code. RGB is an approximate preview for editing, not a calibrated CRT measurement.
Use Picker to open the built-in color picker. Astrocade palette cells set both Byte and Preview; quick swatches/custom hex change only the editor preview. Edit the Byte field separately if you want a specific hardware byte. Use OK or X to keep changes, or Cancel/Esc to revert this dialog session.
