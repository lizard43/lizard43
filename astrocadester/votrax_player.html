Playback notes

Primary target: real Wizard of Wor / Gorf-style arcade hardware. The built-in samples now come from wow.asm phrase blocks where SAYPH outputs each direct SC-01 byte to speech port $17 and GSPST waits on input $12 bit 7. Built and exported phrases should be direct SC-01 $00-$3F byte streams ending with $3F STOP.
This page uses the browser AudioContext and a small formant/noise synthesizer. It is meant for fast Astrocade phrase prototyping from ROM byte streams, not cycle-accurate Votrax SC-01 emulation.
Click any green phoneme pill in the Decoded or phonemes tab to audition it. Use Add when you want to append raw SC-01 phonemes to the phrase.
Hardware/export rule: audio preview code is never written into the exported .DB. For the current WoW/Gorf hardware workflow, export is the same direct SC-01 bytes shown in the editor. The tokens tab stays visible for reference and audition. Direct SC-01 hardware phrases still use the phoneme palette for bytes sent to port $17. Dragging Wizard/WoW tokens into a direct SC-01 phrase is guarded until the original WoW encoded-string decoder is implemented.
The SC-01 chip internally contains its own phoneme timing and articulation behavior. This browser version approximates phoneme length, voiced/unvoiced consonants, glides, fricatives, plosives, and pauses using simplified envelopes. Built-in SC-01 / SC-01A ROM tables are available immediately; use the toolbar ROM dropdown to switch the active 64-entry internal phone ROM used for duration, amplitude, formants, noise, and closures.
